# Implement the next GitHub not-closed issue (with "Planned" label AND without "Human" label AND without "Triage" label AND without "In Progress" label)

## Overview

Take the next GitHub not-closed issue (with "Planned" label AND without "Human" label AND without "Triage" label AND without "In Progress" label) and implement it.

## Steps

1. Check the next GitHub not-closed issue (with "Planned" label AND without "Human" label AND without "Triage" label AND without "In Progress" label). Take high priority issues first.
   - If there is no remaining task OR if remaining tasks have the "Human" or "Triage" or "In Progress" label, STOP at this step with a message "No task found".

2. Display clearly in the output the task you picked (display at least its id and its title).

3. Add the "In Progress" label to the issue.

4. Execute/implement the task.

5. After the implementation is done and acceptance criteria are met, update the "## Implementation plan / notes" section in the GitHub issue with what you have done.

6. Add the "Implemented" label to the task but don't remove the "In Progress" label.
